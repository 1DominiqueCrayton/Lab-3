# Lab 3 — Manage User Account Statuses

## Objective
View and manage user account statuses.

---

## Reset the password for a user

### Scenario
Crystal Williams forgot her password, and you need to reset it.

---

### Step 1
From the Admin Console, view the user account for **Crystal Williams**.

---

### Step 2
Reset the password:

a. Select **Reset or Remove password**  
b. Select the option to **send a reset password email**  
c. Verify that the user will be **signed out of all devices and browsers**  
d. Reset the password
